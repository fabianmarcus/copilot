# Meine persönliche Sammlung von Instructions und Skills

Das ist meine zentrale Sammlung von Anweisungen (Instructions) und Skills für GitHub Copilot und verwandte Tools.

- Ich verwalte diese Sammlung zentral und nutze sie aktiv in mehreren Projekten.
- Sie enthält sowohl selbst erstellte Anleitungen als auch ausgewählte Inhalte aus der Community, insbesondere von [Awesome Copilot](https://github.com/github/awesome-copilot/).
- Mein Ziel ist es, für mich bewährte Praktiken, hilfreiche Workflows und nützliche Erweiterungen an einem Ort zu bündeln.

Ich erweitere und passe die Sammlung regelmäßig an, um aktuelle Entwicklungen und meine eigenen Erfahrungen zu berücksichtigen.
